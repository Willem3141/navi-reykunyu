Contributions to Reykunyu are very welcome! But please keep the following in mind:

* Not all potential ideas are a good fit for Reykunyu. Also, maybe the maintainers have suggestions on how to best implement your idea. So before you start working on anything, please contact the maintainers (for example, by opening an [issue](https://github.com/Willem3141/navi-reykunyu/issues) or a [discussion topic](https://github.com/Willem3141/navi-reykunyu/discussions), or by asking on Discord (see the Reykunyu topic in [Kelutral](https://kelutral.org/)'s `#language-meta` channel).
* Because Reykunyu is developed in limited spare time, please keep in mind that there may be long delays before your contributions are looked at.
* Reykunyu has a strict no-AI policy. Any code that is (co-)written by LLMs is not allowed. The same holds for AI-written or AI-assisted issue/PR descriptions. (Exception: tools for machine translation, such as [DeepL Translator](https://www.deepl.com/en/translator), are acceptable.)

Irayo nìtxan!
